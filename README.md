# MusicHub API

**MusicHub API** — асинхронный веб-сервис для музыкантов и слушателей.
Сервис позволяет публиковать треки, назначать теги, создавать плейлисты, подписываться на авторов и находить музыку по гибким фильтрам.

---

## Возможности

### Пользователи

* регистрация и авторизация
* создание и управление своими треками
* подписка на других пользователей
* получение уведомлений о новых треках/альбомах авторов (email-рассылка)

### Треки

* добавление треков и редактирование их метаданных
* просмотр треков других пользователей
* привязка нескольких тегов к одному треку
* поддержка любых типов тегов:

  * жанр
  * принадлежность к альбому
  * информация о ремиксе
  * автор оригинальной версии
  * пользовательские категории

### Теги

Теги являются универсальной системой классификации.
Комбинация тегов позволяет формировать уникальные выборки треков.

### Плейлисты

* создание плейлистов
* добавление/удаление треков
* автоматическая генерация сложных подборок

  * пример: альбом **«Первый класс»** + жанр **«Русский фолк»**
    → подборка всех ремиксов из этого альбома в указанном жанре

### Поиск

* фильтрация по нескольким тегам
* сортировка (по дате, популярности, автору)
* поиск по названию и описанию
* комбинированные запросы

---

## Архитектура

Сервис построен на событийной архитектуре:

* публикация новых треков генерирует событие в Kafka
* сервис подписок обрабатывает события и отправляет email-уведомления
* Redis используется как кеш для ускорения поиска и хранения горячих данных
* PostgreSQL отвечает за хранение основной модели данных
* SQLAlchemy — ORM-слой
* Alembic — миграции схемы

* **Python**

* **SQLAlchemy** — ORM

* **Alembic** — миграции

* **Kafka** — брокер сообщений и событийная коммуникация
