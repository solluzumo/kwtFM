KWT-FM API
Описание проекта
KWT-FM API — это веб-сервис для музыкантов и любителей музыки, предоставляющий платформу для обмена треками, создания плейлистов и взаимодействия между пользователями. Сервис предлагает расширенные возможности фильтрации, тегирования и персонализированные рекомендации.

Основные возможности
Управление треками
Добавление пользователями своих треков

Назначение тегов трекам (жанр, альбом, тип ремикса и др.)

Просмотр треков других пользователей

Расширенный поиск с фильтрацией и сортировкой

Система тегов
Гибкая система тегирования для описания:

Жанр и поджанр

Принадлежность к альбому

Статус ремикса/кавера

Автор оригинала (для ремиксов)

Год выпуска и другая метаинформация

Социальные функции
Подписка на других пользователей

Email-рассылка при выходе новых треков/альбомов

Персональная лента обновлений

Интеллектуальные плейлисты
Автоматическая генерация уникальных плейлистов на основе тегов

Комбинирование условий (например: альбом "Первый класс" + жанр "Русский фолк")

Подбор ремиксов определенного жанра для конкретного альбома

Технологический стек
Backend
Python 3.11+

FastAPI — веб-фреймворк

SQLAlchemy — ORM

Alembic — управление миграциями БД

Pydantic — валидация данных

Базы данных и кеширование
PostgreSQL — основная база данных

Redis — кеширование, сессии

Kafka — брокер сообщений, обработка событий

Инфраструктура
Docker — контейнеризация

Docker Compose — оркестрация

Nginx — прокси-сервер

JWT — аутентификация
