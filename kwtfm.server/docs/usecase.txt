Неавторизованный пользователь может:
* Зарегистрироваться;
* авторизоваться;
* восстановить пароль.

Авторизованный пользователь может:

------ТРЕК--------
* Загрузить трек на площадку с указанными тегами, авторами и названием;
* удалить свой трек с площадки;
* получить отфильтрованный, отсортированный список треков;
* лайкнуть трек;
* поделиться треком в соцсетях или скопировать ссылку.
* посмотреть подробную информацию о треке.

------ПЛЕЙЛИСТ--------
* просмотреть плейлист;
* создать плейлист;
* добавить трек в плейлист;
* удалить трек из плейлиста;
* поменять порядок треков в плейлисте;
* удалить плейлист;
* добавить трек в основной плейлист;
* сохранить конкретный плейлист;

------ПРОФИЛЬ--------
* редактировать профиль;
* подписаться на другого пользователя;
